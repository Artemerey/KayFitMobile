# KayFit 1.2.1+3 — Changelog

**Дата:** 2026-06-07

## Новое

- **Скидочный оффер** — поддержка RevenueCat-оффера `discount`: при активном оффере карточка тарифа показывает зачёркнутую старую цену и выделяет новую акцентным цветом. Новый `discountOfferingProvider` и `originalPriceLabel` в `PaywallPlanCard`.
- **Автопоказ пейвола в журнале** — при первом открытии экрана «Журнал» для незаплативших пользователей вызывается `maybeShowPaywallOnce`. Флаг `_paywallCheckDone` исключает повторный показ при перестройке виджета.
- **Speech-to-text** — добавлена зависимость `speech_to_text ^6.6.0`; разрешение `NSSpeechRecognitionUsageDescription` в `Info.plist`.
- **Провайдеры состояния навигации** — `transcriptionInProgressProvider` и `chatProcessingProvider` (NOT autoDispose) сохраняют состояние spinner/thinking-bubble при смене вкладок; экран чата восстанавливает индикаторы при возврате.

## Исправления

- **BUG-2 (VOICE-PENDING)** — результат транскрипции больше не теряется при переходе в журнал во время HTTP-запроса к `/api/transcribe`. Причина: `WidgetRef` бросал `AssertionError` (не `Exception`) после `dispose()` виджета, поэтому `on Exception catch` его не ловил. Фикс: `pendingNotifier` захватывается до первого `await`, пока виджет точно mounted.
- **BUG-1 (VOICE-RESET)** — флаг `_fromVoice` сбрасывается при ручном редактировании текста, предотвращая ложную передачу `is_voice=true` при обычном вводе.
- Дублирующий автопоказ пейвола устранён: guard `_paywallAutoShowDone` предотвращает повторный вызов.
- Скидочный UI скрыт когда реальное изменение цены отсутствует.
- Локализация пустого состояния журнала (RU/EN).

## Локализация

- Полная локализация: все хардкодированные EN-строки (П/Ж/У → Б/Ж/У, KCAL → ккал, метки чата и журнала) переведены через ARB-ключи.
- Дефолтный язык = язык устройства (`deviceLocale`); явный выбор сохраняется флагом `app_locale_user_set`.

## Технические изменения

- `parse_meal.py` (backend patch): системный промпт упрощён — удалена билингвальная таблица порций, заменена компактными дефолтами (основное 200 г, суп 300 г, напиток 250 г, гарнир 150 г). Составные названия блюд — одна позиция.
- Флаг `is_voice` передаётся в `/api/parse_meal` при голосовом вводе.
- 0 предупреждений `dart analyze` (было 51).
- Регрессионные тесты для BUG-1 и BUG-2.
